# Picture Diet

Picture Diet е малък инструмент за фотографи, създаден с една цел – бързо и лесно намаляване размера на голям брой JPG/JPEG снимки с минимална загуба на качество.

Програмата обработва автоматично всички снимки в избрана папка и записва компресираните файлове в отделна директория, без излишни настройки и сложни менюта.

## Основни функции

* Избор на входяща и изходяща папка
* Автоматична обработка на JPG/JPEG файлове
* Оптимизирано JPEG компресиране
* Прогрес бар за следене на процеса
* Лог с информация за обработените файлове
* Изчистен тъмен интерфейс
* Работа във фонов режим без блокиране на интерфейса

## Как работи

1. Изберете папка със снимки.
2. Изберете папка за записване на компресираните файлове.
3. Натиснете **Стартирай**.
4. Изчакайте обработката да приключи.
